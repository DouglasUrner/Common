
## Washington GLEs



## Objectives

* Track changes to your code.
* Collaborate with a team on a coding project.
* Make mistakes and recover from them.

## Essential Skills

* Create a repository
* Make a commit
* Push
* Pull
* Create and switch branches
* Merge branches
* Resolve conflicts
* Revert a commit

Plus - seaching for how to do something you don't know/remember.
